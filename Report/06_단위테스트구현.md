### TC_16 Red 원인

```cpp
// SHealth.cpp — README는 >= 25 비만
if (bmi > kBmiOverweightMax) {  // 25.0은 None 또는 과체중
    return BmiClassSlot::Obesity;
}
```

**Green 턴:** `bmi >= kBmiOverweightMax`로 수정 → `tc16` weight `72.25` 재검증 → TC_18 회귀 확인.
