## 요약

README와 현행 `SHealth` 구현을 C++·테스트 관점에서 정리했다. **핵심 QA 이슈**는 (1) BMI=25 경계 미분류, (2) `calculateBmi` 단일 God Method로 단위 테스트 곤란, (3) `ageCount==0`·`sum==0` 0 나눗셈, (4) GTest 스텁만 존재한다는 점이다. 2~3단계 리팩토링·UT 작성 시 본 문서 §4 TC 1~39번을 우선 구현 기준으로 사용한다.

## 2. BMI 계산 로직 이해

### 2.1 수식

```
height_m = height_cm / 100.0
BMI      = weight_kg / (height_m * height_m)
```

### 2.2 처리 파이프라인

```mermaid
flowchart LR
    A[CSV 로드] --> B[연령대별 weight=0 보정]
    B --> C[BMI 계산]
    C --> D[연령대별 4분류 집계]
    D --> E[백분율 저장]
```

| 단계 | 내용 | 리스크 |
|------|------|--------|
| 로드 | `split(',', …)` → age, weight, height | `stoi`/`stod` 예외 미처리 |
| 보정 | 20~70, 10년 단위 2-pass 평균 | `ageCount==0` → 0 나눔 |
| BMI | 보정 후 전 행 계산 | `height==0` → inf/NaN |
| 통계 | 구간별 분류 카운트 → % | `sum==0`, BMI=25 갭 |

### 2.3 분류 의사결정 (요구 명세·권장 C++)

| 분류 | 조건 | 권장 표현 |
|------|------|-----------|
| 저체중 | BMI ≤ 18.5 | `bmi <= 18.5` |
| 정상체중 | 18.5 < BMI < 23 | `bmi > 18.5 && bmi < 23.0` |
| 과체중 | 23 ≤ BMI < 25 | `bmi >= 23.0 && bmi < 25.0` |
| 비만 | BMI ≥ 25 | `bmi >= 25.0` |

### 2.4 `shealth.dat` 검증 포인트

| id | age | weight | height | 비고 |
|----|-----|--------|--------|------|
| 93730 | 57 | **0** | 167.6 | F-03 보정 대상 |
| 93717 | 22 | 74.4 | 174.1 | 정상 분류 후보 |
| 93709 | 75 | 88.8 | 151.1 | 20–70 루프 밖 |

## 4. Google Test 테스트 시나리오 (번호)

> 현재: `TEST(SHealthBMITest, FailedTest) { FAIL(); }` — **전부 작성 대상**  
> 네이밍: `TEST_F(SHealthFixture, TC_nn_...)`, Given-When-Then 주석

### 4.1 BMI 계산

1. weight=70, height=170 → BMI ≈ 24.22 (`EXPECT_NEAR`, ε=1e-2)
2. height=100cm → BMI = weight
3. README 샘플 (79.5, 158.3) 재현
4. 큰 height — 유한 `double`
5. height=0 — F-10·예외 정책 확정 후 TC

### 4.2 체중 보정 (F-03)

6. 동대 {50, 60, 0} → 0행은 55.0
7. 동대 전원 weight=0 — `ageCount==0` 방어
8. 20대 평균이 30대에 적용되지 않음
9. 유효 1명·누락 1명 → 평균=유효 체중
10. 보정 전후 BMI 변경 검증

### 4.3 BMI 분류 경계 (F-05)

11. BMI=18.5 → 저체중  
12. BMI=18.5000001 → 정상  
13. BMI≈22.999 → 정상  
14. BMI=23.0 → 과체중  
15. BMI≈24.999 → 과체중  
16. **BMI=25.0 → 비만** (현 구현 Red 예상)  
17. BMI=30.0 → 비만  
18. 임의 표본 — 정확히 1개 분류

### 4.4 연령대 통계 (F-06, F-07)

19. 소규모 CSV, 20대 4명 1:1:1:1 → 각 25%  
20. 해당 연령대 0명 — 0% 또는 N/A  
21. `getBmiRatio(20, 100)` = 20대 저체중 %  
22. 잘못된 ageClass/type → 0.0 또는 에러  
23. `calculateBmi` 전 `getBmiRatio` → 0.0

### 4.5 파일·파싱 (F-01)

24. 없는 경로 → count 0  
25. 헤더만 → count 0  
26. 빈 줄·토큰 부족 — 중단/skip 정책  
27. `shealth.dat` 스모크 — count>0, 유한 비율  
28. id 무시 — age/weight/height만으로 동일

### 4.6 예외·비기능

29. 레코드 >10000 — 오버플로/거부  
30. 음수 age/weight/height 정책  
31. `calculateBmi` 2회 호출 — 통계 덮어쓰기  
32. `istringstream` 픽스처 — 디스크 없이 반복

### 4.7 로드맵 연계 (F-09~F-12, Green 후)

33. height=0 평균 보정 (F-10)  
34. 정상 BMI 사용자 목록 (F-11)  
35. 전체 4분류 비율 합 ≈ 100% (F-12)  
36. 연령대별 4분류 합 ≈ 100%

### 4.8 인프라

37. `FailedTest` 제거, `ctest` Green  
38. `test/fixtures/minimal.csv` 등 소형 픽스처  
39. `TEST_F` + `std::filesystem` 임시 경로
